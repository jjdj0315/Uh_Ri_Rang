## 2. 리더보드 디자인 개선

### 2.1 상위권 유저 강조
- [ ] **Top 3 포디엄 디자인**
  - 1등: 금색 트로피 아이콘 + 큰 카드/강조 배경
  - 2등: 은색 트로피 + 중간 크기
  - 3등: 동색 트로피 + 기본 크기
  - 현재 이모지(🥇🥈🥉) → 아이콘 + 배경 색상 + 크기 차등으로 업그레이드
- [ ] 상위 3팀은 테이블 위에 별도 카드 영역으로 분리 (포디엄 스타일)

### 2.2 내 위치 시각화
- [ ] **하이라이트 행**: 내 팀(userProfile.teamName)과 일치하는 행을 강조 색상으로 표시
- [ ] **내 순위 요약 카드** (테이블 위)
  - "내 팀: 404found | 순위: 1위 | 점수: 87.5"
  - 전체 참가팀 대비 상위 몇 %인지 표시 (예: "상위 12%")
- [ ] **순위 분포 바**: 전체 점수 분포에서 내 위치를 시각적으로 표시
  - 가로 바 형태, 내 위치에 마커 표시
